# Как посмотреть данные в PgHero

## Что показывает "Explain":

Раздел "Explain" показывает **план выполнения запроса**, а не сами данные. Это инструмент для оптимизации, а не для просмотра данных.

**Что означает результат:**
- `Seq Scan on users` - последовательное сканирование таблицы users
- `cost=0.00..17.20` - стоимость запроса (в условных единицах)
- `rows=720` - ожидаемое количество строк
- `width=84` - средний размер строки в байтах

Это НЕ сами данные, а информация о том, как PostgreSQL будет выполнять запрос.

---

## Как посмотреть САМИ данные в PgHero:

### Проблема:
PgHero в основном предназначен для мониторинга и анализа, а не для просмотра данных через GUI.

### Решение 1: Используйте DBeaver (РЕКОМЕНДУЕТСЯ) ⭐⭐⭐

**Самый простой способ:**

1. Скачайте DBeaver: https://dbeaver.io/download/ (бесплатно)
2. Создайте подключение к PostgreSQL через External Database URL
3. Просматривайте таблицы через удобный графический интерфейс

**Подробная инструкция:** См. файл `QUICK_DBEAVER_SETUP.md`

---

### Решение 2: Используйте psql (командная строка)

```bash
# Подключитесь к базе
psql "postgresql://user:password@host:port/database"

# Выполните запросы
SELECT * FROM users;
SELECT * FROM logs;
```

---

### Решение 3: Попробуйте другие разделы PgHero

В некоторых версиях PgHero может быть раздел для выполнения SQL запросов:

1. Проверьте раздел **"Tune"** - там может быть SQL редактор
2. Или используйте любой раздел, где можно ввести SQL

Но обычно PgHero не предназначен для просмотра данных - это инструмент мониторинга.

## Что можно делать в PgHero:

### ✅ Полезные функции PgHero:

1. **Overview** - общая статистика и здоровье БД
2. **Space** - использование места
3. **Connections** - активные подключения
4. **Live Queries** - выполняемые запросы в реальном времени
5. **Maintenance** - обслуживание БД
6. **Explain** - анализ планов выполнения запросов
7. **Tune** - рекомендации по оптимизации

### ❌ Что НЕ может PgHero:

- Удобный просмотр таблиц через GUI
- Редактирование данных через интерфейс
- Прямой просмотр содержимого таблиц

---

## Рекомендация:

**Для просмотра и изменения данных используйте DBeaver** - это самый удобный бесплатный инструмент с графическим интерфейсом.

**PgHero оставьте для мониторинга** - он отлично подходит для:
- Отслеживания производительности
- Поиска медленных запросов
- Мониторинга использования ресурсов
- Анализа индексов

---

## Быстрый старт с DBeaver:

1. Скачайте: https://dbeaver.io/download/
2. Установите и запустите
3. New Database Connection → PostgreSQL
4. Используйте External Database URL из Render
5. Готово! Теперь можете просматривать таблицы и данные

Подробнее в файле `QUICK_DBEAVER_SETUP.md`
