# САМЫЙ ПРОСТОЙ способ - через бота в Telegram! ⭐

## Просто используйте команды бота:

Все данные можно посмотреть через бота в Telegram:

- `/me` - ваш прогресс (всего, сегодня, среднее)
- `/top` - лидерборд всех участников
- `/today` - подтягивания за сегодня

**Это работает прямо сейчас и показывает все данные!**

---

## Если все-таки нужен прямой доступ к базе:

### Вариант 1: Установите psql (5 минут)

```bash
# macOS
brew install postgresql

# Затем подключитесь
psql "postgresql://challenge_user:password@dpg-xxxxx-a.frankfurt-postgres.render.com:5432/challenge_bot"
```

Где взять URL:
- Render Dashboard → PostgreSQL → Info → External Database URL
- Скопируйте весь URL целиком

### Вариант 2: DBeaver - используйте вкладку URL

1. New Database Connection → PostgreSQL
2. **Переключитесь на вкладку "URL"** (не Main!)
3. Вставьте **весь External Database URL** целиком
4. Test Connection

---

## Что вы хотите сделать с данными?

- **Просто посмотреть?** → Используйте команды бота (`/me`, `/top`)
- **Изменить данные?** → Можете добавить через бота (`/add`)
- **Удалить запись?** → Используйте `/undo`
- **Нужен прямой SQL доступ?** → Установите psql (см. выше)

**Для большинства задач команды бота достаточно!**
